## Development

```bash
npm install
npm run dev
```

로컬 서버: [http://localhost:3030](http://localhost:3030)

캐시 초기화 후 실행:

```bash
npm run dev:clean
```

---

## Build

```bash
npm run build
```

GitHub Pages 배포 시 `main` 브랜치 push 후 Actions 워크플로가 자동 빌드·배포합니다.

로컬에서 Pages 경로로 미리보기하려면:

```bash
NEXT_PUBLIC_BASE_PATH=/portflio2026 NEXT_PUBLIC_SITE_URL=https://kararico.github.io/portflio2026 npm run build
npx serve out
```
