불스원

불스원은 자동차 관리 제품과 브랜드 콘텐츠를 제공하는 웹사이트입니다. 제품 카테고리가 다양해 사용자가 원하는 정보에 도달하기까지 탐색 단계가 길었고, 모바일·데스크톱 간 UI 경험이 일관되지 않으면 브랜드 신뢰도에 영향을 줄 수 있는 구조였습니다. 제품 탐색과 브랜드 스토리 전달이 자연스럽게 이어지는 UI가 필요했습니다.

  • 반응형 레이아웃·인터랙션 구현 — 모바일·데스크톱 간 동일한 브랜드 경험
  • 공통 UI 컴포넌트 정리 — 페이지 추가·수정 시 UI 일관성 유지
  • 브랜드 스토리·제품 정보 영역의 콘텐츠 흐름 설계

Outcome

제품 탐색과 브랜드 콘텐츠 소비가 끊기지 않도록 화면 구조를 개선하고, 30개 이상 제품 페이지를 공통 UI 기준으로 정리했습니다. 운영·기획 변경에도 UI 품질 편차 없이 대응할 수 있는 컴포넌트 기반을 남겨, 장기적인 유지보수 효율을 확보했습니다.
