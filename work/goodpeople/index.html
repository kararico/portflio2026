좋은사람들

좋은사람들은 다양한 패션·언더웨어 브랜드를 하나의 서비스에서 운영하는 브랜드 통합 쇼핑몰입니다. 브랜드별 상품과 프로모션을 일관된 사용자 경험 안에서 제공하면서도 각 브랜드의 아이덴티티를 유지할 수 있는 구조가 필요했습니다. 상품 탐색부터 상세, 주문까지 통합 커머스 환경에 맞는 UI 구축과 퍼블리싱을 담당했습니다.
