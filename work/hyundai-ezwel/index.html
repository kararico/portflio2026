현대이지웰

현대이지웰은 기업 임직원이 복지·제휴 서비스를 한곳에서 탐색·이용하는 복지 플랫폼입니다. 서비스 카테고리와 제휴 콘텐츠가 계속 늘어나면서, 사용자는 원하는 혜택을 찾기 어렵고 운영 측은 화면마다 UI 기준이 달라지는 문제가 있었습니다. 서비스 확장에도 흔들리지 않는 탐색 경험과 유지보수 가능한 UI 기반이 필요했습니다.

Role

PL · 복지몰 플랫폼 UI 설계 및 퍼블리싱

Contribution

  • Vue.js 기반 컴포넌트·레이아웃 구조 설계 — 복지 서비스 추가 시에도 동일 UI 패턴 적용
  • 복지 서비스 탐색·카테고리·상세 흐름의 정보 구조 정리 — 사용자 혼선을 줄이는 UI 위계 설계
  • 웹표준·웹접근성·크로스 디바이스 QA 기준 수립 및 적용
  • 운영·기획 변경에 대응 가능한 컴포넌트 단위 UI 구조로 유지보수성 확보

Outcome

플랫폼 핵심 화면을 공통 UI 구조 위에 구축해, 복지 서비스가 늘어나도 탐색·이용 경험이 일관되게 이어지도록 했습니다. PL로서 퍼블리싱 품질 기준을 정립해 이후 화면 작업과 QA에 동일한 기준을 적용할 수 있게 했고, 운영팀이 콘텐츠·서비스 변경에 대응하기 쉬운 UI 기반을 남겼습니다.
